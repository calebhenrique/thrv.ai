# Guia de Contribuição

Obrigado por considerar contribuir para o Gem Engenheiro de Prompts! Este documento fornece diretrizes e instruções para contribuir.

## Como Contribuir

### Reportar Bugs

Se encontrou um bug:

1. Verifique se o bug já foi reportado em Issues
2. Se não, abra uma nova Issue com:
   - Título descritivo
   - Descrição clara do problema
   - Passos para reproduzir
   - Comportamento esperado vs atual
   - Screenshots ou exemplos se aplicável

### Sugerir Melhorias

Para sugerir melhorias:

1. Descreva a melhoria em detalhes
2. Explique por que seria útil
3. Liste exemplos de uso
4. Considere impactos e alternativas

### Pull Requests

1. **Fork o repositório**
